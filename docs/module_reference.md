# Module Reference

## Core Modules

### Scanner Engine (`core/scanner_engine.py`)
Main coordinator untuk seluruh scanning process.

**Key Methods:**
- `start_scan(url)` - Main scanning entry point
- `advanced_discovery_phase()` - Path discovery
- `intelligence_analysis_phase()` - Data analysis

### Session Manager (`core/session_manager.py`)
Manage HTTP sessions dan cookies.

**Features:**
- Cookie management
- Request history tracking
- Session persistence

### Evasion Engine (`core/evasion_engine.py`)
Anti-detection dan blocking avoidance.

**Techniques:**
- User-Agent rotation
- Request timing randomization
- Blocking detection

## Intelligence Modules

### ML Predictor (`intelligence/ml_predictor.py`)
Machine Learning untuk path prediction.

**Capabilities:**
- Learn from successful scans
- Generate smart path variations
- Pattern recognition

### NLP Analyzer (`intelligence/nlp_analyzer.py`)
Natural Language Processing untuk content analysis.

**Detection:**
- Secrets & credentials
- Technology stacks
- API endpoints

### Adaptive Learner (`intelligence/adaptive_learner.py`)
Real-time learning dan optimization.

**Features:**
- Performance tracking
- Optimization suggestions
- Pattern analysis

### Intelligent Crawler (`discovery/intelligent_crawler.py`)
