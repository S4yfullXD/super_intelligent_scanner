# Generated Reports

This directory contains generated reports from scans.

## Report Types

### HTML Reports (`detailed_report.html`)
- Visual overview of scan results
- Risk assessment and recommendations
- Easy-to-read format for sharing

### JSON Reports (`scan_report.json`)  
- Machine-readable data
- Complete scan metadata
- Suitable for further analysis

## Usage
- Open HTML reports in web browser
- Use JSON reports for data processing
- Compare reports across different scans
